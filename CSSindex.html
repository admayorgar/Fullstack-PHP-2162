Introduccion a CSS

CSS [Cascading Style sheets] Hojas de estilo en cascada
Las instrucciones se ejecutan de arriba hacia abajo
Hay tres maneras de escribir estilos.
Una buena practica es escribirlos afuera de html - tercera opción ETIQUETA QUE USAMOS SIEMPRE: style


PRIMERA FORMA DE DAR ESTILO

ESCRIBO ESTO EN LO QUE SERÍA EL HEAD
Agrego la etiqueta style que abre y cierra, y dentro:
h1 {color: red; /* Propiedad: Valor; */ }
La forma de comentar dentro del estilo es: /*Comentarios en CSS */

SEGUNDA FORMA DE DAR ESTILO

En este caso estoy usando la etiqueta de parrafo, dentro del comienzo, agrego el style = "color : choose your color " ;

SEGUNDA FORMA DE DAR ESTILO

Creo una carpeta dentro de mi proyecto para los estilos. Los archivos los guardaré con la extensión .css
Ahi sin necesidad de abrir style, escribo lo que necesito y guardo.
Para poder linkear los dos archivos, despues del 3er escribo la etiqueta "link" y presiono tab o enter, para agregar el link del archivo En href"" -> Ctrol+Spacebar para ubicar la carpeta y seleccionar el doc, o simplemente cuando empiezo a escribir el nombre de mi archivo css me lo autocompletará
